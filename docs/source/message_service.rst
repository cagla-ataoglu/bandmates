Message Service Module
======================

This module facilitates chat and message functionalities in an application, leveraging AWS DynamoDB to store and manage chat rooms, chat members, and messages.

MessageService Class
--------------------

The ``MessageService`` class provides methods to create chats, send messages, and retrieve chat histories and messages. This includes creating chat rooms, adding members to chats, and managing message sending and retrieval.
