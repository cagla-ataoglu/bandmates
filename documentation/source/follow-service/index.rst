Follow Service Module
=====================

This module contains the ``FollowService`` class, which manages follow relationships between users in a DynamoDB database. It includes functionalities to handle follow requests, create follow connections, and query followers and followings.

Follow Service Class
--------------------

The ``FollowService`` class handles the creation and management of follow relationships stored in DynamoDB tables. It provides methods for sending follow requests, creating follow connections, and retrieving follow-related data.
