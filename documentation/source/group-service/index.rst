Group Service Module
====================

The Group Service module manages the creation, modification, and querying of groups, posts, and group memberships.

GroupService Class
------------------

The ``GroupService`` class handles operations related to groups. It provides methods for creating and deleting groups, editing group details, managing the members and the posts of a group.
