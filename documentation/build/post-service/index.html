Post Service Module¶

This module provides functionalities for creating, retrieving, updating, and deleting posts within a social network application, utilizing DynamoDB for storage and S3 for media hosting.

PostService Class¶

The PostService class includes methods for handling post-related operations such as adding new posts, retrieving posts by ID, updating post descriptions, deleting posts, and managing post media via AWS S3.
