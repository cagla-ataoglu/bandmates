Auth Service Module¶

This module provides a service for managing user authentication and registration using AWS Cognito. It includes functionalities such as user signup, password management, and token validation.

CognitoService Class¶

The CognitoService class encapsulates the interaction with AWS Cognito. It supports operations such as signing up new users, changing passwords, authenticating users, and validating JWTs for secure service access.
